Decorator

CONTEXT

This is a Java program that simulates decorating a room. The player will need to clean an empty messy room and redecorate it to make it comfortable and cosy. There will be instruction on the top-left corner of screen, the player can interact with this program by mouse clicking and dragging.

Goal and Requirements Oriented

When creating a game, it is important to make the workflow smooth and rational, which means there should be appropriate instructions for players so that they will not get lost. When the whole workflow is determined and polished, the next step is to achive these different functions and effects by coding. At last, proper sound effects also help smooth the whole game wrokflow by providing sound feedback in interaction.

Designing and Decision

The designing in game development can be divided into two parts, which are interface design and code design. For interface design, it is all about how to help players understand the situation better so that they will not get lost in the game. For code design, it is really important to create a logic, simple and straightforward workflow, which is easier to fix and maintain.
